## 🧩 **Day 16 – Q2: Sum of Square Numbers**

**Difficulty:** Medium  
**Topic:** Math  
**Tags:** Math, Two Pointers, Binary Search

---

## 📄 Problem Statement

Given a non-negative integer `c`, decide whether there're two integers `a` and `b` such that `a² + b² = c`.

---

## 🧠 Examples

### Example 1:

Input: c = 5

Output: true

Explanation: 1 * 1 + 2 * 2 = 5

### Example 2:

Input: c = 3

Output: false

---

## ✅ Constraints

- `0 <= c <= 2³¹ - 1`
